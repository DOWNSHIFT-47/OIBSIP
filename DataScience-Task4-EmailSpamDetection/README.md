# 📧 Email Spam Detection using Machine Learning

## 📌 Project Overview

This project classifies SMS messages as **Spam** or **Ham (Not Spam)** using Machine Learning and Natural Language Processing (NLP).

---

## 🎯 Objective

Develop a machine learning model capable of accurately identifying spam messages from legitimate messages.

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- NLTK
- TF-IDF Vectorizer
- Jupyter Notebook

---

## 📂 Dataset

SMS Spam Collection Dataset

The dataset contains:

- SMS Message
- Label (Spam / Ham)

---

## 📊 Exploratory Data Analysis

The following analyses were performed:

- Dataset Inspection
- Data Cleaning
- Missing Value Analysis
- Spam vs Ham Distribution
- Message Length Analysis

---

## 🤖 Machine Learning Models

- Multinomial Naive Bayes
- Logistic Regression

---

## 📈 Evaluation Metrics

- Accuracy
- Confusion Matrix
- Precision
- Recall
- F1-Score
- Classification Report

---

## 📋 Model Performance

| Model | Accuracy |
|--------|---------:|
| Multinomial Naive Bayes | **96.86%** |
| Logistic Regression | **94.44%** |

---

## 🏆 Best Model

The **Multinomial Naive Bayes** classifier achieved the highest accuracy (**96.86%**) and demonstrated better overall performance for spam message classification.

---

## 📌 Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Exploratory Data Analysis
5. Text Vectorization using TF-IDF
6. Train-Test Split
7. Model Training
8. Model Evaluation
9. Model Comparison
10. Conclusion

---

## 📌 Conclusion

The project successfully classified SMS messages into Spam and Ham categories using Natural Language Processing and Machine Learning techniques.

Among the evaluated models, the **Multinomial Naive Bayes** classifier achieved the highest accuracy (**96.86%**) and was selected as the best-performing model.
