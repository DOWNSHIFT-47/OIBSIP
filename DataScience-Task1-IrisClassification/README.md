# 🌸 Iris Flower Classification

## 📌 Project Overview
This project aims to classify Iris flowers into three species—**Setosa**, **Versicolor**, and **Virginica**—using Machine Learning classification algorithms. The classification is based on four flower measurements:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The project demonstrates the complete machine learning workflow, including data loading, exploratory data analysis (EDA), visualization, model training, evaluation, and comparison.

---

## 🎯 Objective
Develop a machine learning model that accurately identifies the species of an Iris flower using its physical measurements.

## 📌 Conclusion

Both Logistic Regression and Random Forest achieved **100% accuracy** on the Iris dataset.

Based on the evaluation, **Random Forest** was selected as the best-performing model due to its robustness and strong classification performance.

The analysis also showed that **Petal Length** and **Petal Width** are the most significant features for classifying Iris flower species.
