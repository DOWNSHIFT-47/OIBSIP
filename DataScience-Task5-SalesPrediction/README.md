# 📈 Sales Prediction Using Machine Learning

## 📌 Project Overview

This project focuses on predicting product sales based on advertising expenditure across different marketing channels. Machine learning regression algorithms are used to analyze the relationship between advertising budgets and sales, enabling businesses to make data-driven marketing decisions.

---

## 🎯 Objective

To develop a machine learning model that accurately predicts product sales using advertising expenditures on:

- 📺 TV
- 📻 Radio
- 📰 Newspaper

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 📂 Dataset

The dataset contains advertising expenditure and corresponding sales information.

### Features

- TV
- Radio
- Newspaper

### Target Variable

- Sales

---

## 📊 Exploratory Data Analysis (EDA)

The following analyses were performed:

- Dataset Inspection
- Data Types
- Missing Value Analysis
- Descriptive Statistics
- Pair Plot
- Correlation Heatmap
- Sales vs TV Visualization
- Sales vs Radio Visualization
- Sales vs Newspaper Visualization

---

## 🤖 Machine Learning Models Used

1. Linear Regression
2. Random Forest Regressor

---

## 📈 Evaluation Metrics

The models were evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## 📋 Model Performance

| Model | MAE | RMSE | R² Score |
|------|------:|------:|------:|
| Linear Regression | **1.465** | **1.789** | **0.899** |
| Random Forest Regressor | **0.670** | **0.824** | **0.978** |

---

## 🏆 Best Model

The **Random Forest Regressor** outperformed Linear Regression by achieving:

- Lower Mean Absolute Error (MAE)
- Lower Root Mean Squared Error (RMSE)
- Higher R² Score

Therefore, **Random Forest Regressor** was selected as the final model for this project.

## 📌 Conclusion

This project successfully demonstrated how machine learning can predict product sales based on advertising expenditure.

The analysis revealed that **TV advertising has the strongest influence on sales**, followed by **Radio**, while **Newspaper advertising has the least impact**.

Among the evaluated models, the **Random Forest Regressor** achieved the best predictive performance with an **R² Score of 0.978**, making it the most suitable model for this dataset.
